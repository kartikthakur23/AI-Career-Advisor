# AI Career Advisor

## 📌 Overview

AI Career Advisor is a Python web application built with Streamlit that recommends suitable career paths based on a user's skills. It uses Natural Language Processing (NLP) techniques such as TF-IDF Vectorization and Cosine Similarity to compare user skills with predefined career requirements.

---

## 🚀 Features

- Skill-based career recommendations
- Interactive Streamlit web interface
- TF-IDF Vectorization for text processing
- Cosine Similarity for matching skills
- Easy-to-use interface
- Instant recommendation results

---

## 🛠️ Technologies Used

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/kartikthakur23/AI-Career-Advisor.git
```

Go to the project folder

```bash
cd AI-Career-Advisor
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
streamlit run app.py
```

---

## 💡 How It Works

1. Enter your skills.
2. The application converts the skills into numerical vectors using TF-IDF.
3. Cosine Similarity compares your skills with different career profiles.
4. The career with the highest similarity score is recommended.
